# Exercise x All Outcomes — Scientific Consensus Reference

## How to Use This Document

This is a self-contained universal reference for exercise effects across 32 health outcomes, organized by outcome domain. Each outcome uses a standardized template with two-tier evidence grading, bias-corrected effect sizes, and dose-response data in MET-h/wk units. Use this document for evidence-based exercise prescription and as a truth table for any exercise-health claim. This document is NOT personalized — see separate research for individual context.

---

## Evidence Grading System

### Tier 1: RCT-Based Evidence (causal inference possible)

| Grade | Criteria |
|-------|----------|
| **A-RCT** | Multiple large RCTs or Cochrane-quality meta-analysis of RCTs; pooled n > 1,000; consistent results; low risk of bias |
| **B-RCT** | 1-2 RCTs or small meta-analysis of RCTs; pooled n < 1,000; some inconsistency or moderate bias risk |
| **C-RCT** | Single small RCT (n < 100) or pilot study; high bias risk; unreplicated |

### Tier 2: Observational Evidence (association, not causation)

| Grade | Criteria |
|-------|----------|
| **A-Obs** | 3+ large prospective cohorts or pooled analyses; consistent results across populations; dose-response present; total n > 100,000 |
| **B-Obs** | 1-2 cohorts or meta-analysis of observational studies; some inconsistency; moderate confounding concerns |
| **C-Obs** | Cross-sectional data, small cohorts (n < 1,000), major uncontrolled confounders, or high reverse causation risk |
